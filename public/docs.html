Pengantar

Selamat datang di dokumentasi API resmi layanan kami. API ini menyediakan antarmuka terprogram untuk mengakses berbagai fitur dan layanan yang kami sediakan.

Dokumentasi ini menjelaskan cara menggunakan API kami dengan contoh permintaan dan respons dalam format JSON. Anda tidak bisa langsung mengakses endpoint API yang memerlukan API Key melalui browser karena membutuhkan otorisasi khusus.

Base URL 

https://www.v-pedia.web.id

Autentikasi

Semua permintaan API yang disebutkan dalam dokumentasi ini memerlukan autentikasi menggunakan API Key Anda. API Key harus disertakan melalui header X-APIKEY. 

Header: X-APIKEY: YOUR_APIKEY
Contoh Penggunaan dengan Header X-APIKEY (untuk endpoint GET /api/profile)
cURL
curl -X GET \
  'https://www.v-pedia.web.id/api/profile' \
  -H 'X-APIKEY: YOUR_APIKEY'
JavaScript (Fetch API)
fetch('https://www.v-pedia.web.id/api/profile', {
  method: 'GET',
  headers: {
    'X-APIKEY': 'YOUR_APIKEY'
  }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
Python (requests)
import requests

headers = {
    'X-APIKEY': 'YOUR_APIKEY'
}
response = requests.get('https://www.v-pedia.web.id/api/profile', headers=headers)
print(response.json())

Penting: Untuk keamanan, beberapa endpoint API Anda mungkin memerlukan IP whitelist yang diatur di panel akun Anda. Pastikan IP yang Anda gunakan untuk mengakses API sudah terdaftar. Jika IP tidak terdaftar, permintaan akan ditolak.

Contoh Response Error

Berikut adalah beberapa contoh respons error yang mungkin Anda temui saat menggunakan API kami, beserta kemungkinan penyebabnya.

API Key Tidak Valid (Status 401 Unauthorized)

{
  "success": false,
  "message": "Invalid API Key"
}

API Key Tidak Disertakan (Status 400 Bad Request)

{
  "success": false,
  "message": "API Key harus disertakan di header \"X-APIKEY\""
}

Akun Belum Terverifikasi (Status 403 Forbidden)

{
  "success": false,
  "message": "Akun belum terverifikasi. Silakan verifikasi akun terlebih dahulu."
}

IP Tidak Diizinkan (Whitelist) (Status 403 Forbidden)

{
  "success": false,
  "message": "IP Anda saat ini tidak diizinkan untuk menggunakan API Key ini.",
  "your_ip": "XXX.XXX.XXX.XXX"
}

Request Terlalu Cepat (Rate Limit) (Status 429 Too Many Requests)

{
  "success": false,
  "message": "Anda harus menunggu 5 detik sebelum melakukan request lagi."
}

Parameter Wajib Tidak Terisi (Status 400 Bad Request)

{
  "success": false,
  "message": "Parameter [nama_parameter] wajib diisi"
}

Saldo Tidak Cukup (Status 400 Bad Request)

{
  "success": false,
  "message": "Saldo tidak mencukupi"
}

Produk Tidak Ditemukan (Status 404 Not Found)

{
  "success": false,
  "message": "Produk tidak ditemukan atau harga tidak valid"
}

Transaksi Tidak Ditemukan (Status 404 Not Found)

{
  "success": false,
  "message": "Transaksi tidak ditemukan atau gagal mendapatkan status"
}

Internal Server Error (Status 500 Internal Server Error)

{
  "success": false,
  "message": "Internal server error"
}
